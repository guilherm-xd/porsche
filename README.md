# Porsche - Não Existem Substitutos

Uma landing page moderna e animada inspirada na identidade visual da Porsche, desenvolvida utilizando apenas HTML e CSS. O projeto foi criado com foco em animações avançadas, efeitos visuais modernos e experiência visual premium, explorando o uso de CSS Keyframes sem JavaScript.

---

## Sobre o Projeto

Este projeto consiste em uma página temática da Porsche desenvolvida para a atividade de CSS Keyframes e animações avançadas.

A proposta foi criar uma interface moderna inspirada em sites automotivos premium, utilizando efeitos neon, fundo animado, loader personalizado, carrossel automático e diversos efeitos visuais feitos exclusivamente com HTML e CSS.

O objetivo principal foi praticar animações profissionais utilizando `@keyframes`, além de desenvolver uma landing page visualmente impactante e responsiva sem o uso de JavaScript ou frameworks externos.

---

## Funcionalidades

* Fundo animado com gradientes e efeitos luminosos
* Texto neon animado
* Loader personalizado com rotação
* Carrossel automático usando apenas CSS
* Imagens com efeito flutuante
* Botões com efeito pulsante
* Hover animado nos cards
* Menu com animações suaves
* Efeito glow em títulos
* Layout responsivo
* Interface inspirada em sites premium automotivos

## Tecnologias Utilizadas

* HTML
* CSS

---

## Animações Utilizadas

O projeto utiliza diversas animações avançadas feitas com `@keyframes`, incluindo:

* `translateY()` para efeito flutuante
* `translateX()` para movimentação do carrossel
* `scale()` para efeitos de zoom e pulsação
* `rotate()` para loader animado
* `text-shadow` para efeito neon
* animações de brilho e iluminação
* animações de background dinâmico
